
# 🌿 Karst Ecological Network Optimization

### Ecological Network Construction and Optimization under Rocky Desertification Disturbance

*A Case Study of Qujing City, Yunnan Province, China*

---

## Overview

Karst ecosystems are highly vulnerable to habitat fragmentation and rocky desertification, threatening biodiversity conservation and ecological resilience.

This project develops an integrated ecological network framework that incorporates rocky desertification sensitivity into ecological source identification, resistance surface construction, corridor extraction, and network optimization.

The framework combines landscape ecology, ecosystem service assessment, graph theory, and spatial ecological modelling to support conservation planning in environmentally fragile karst landscapes.

---

## Research Question

**How can ecological networks be optimized to maintain landscape connectivity and ecological resilience under rocky desertification disturbance?**

## Key Innovations

### Multi-Criteria Ecological Source Identification

Integrated morphological structure, habitat quality, landscape connectivity, and rocky desertification sensitivity to identify ecologically important source areas.

### Rocky-Desertification-Informed Resistance Surface

Incorporated rocky desertification sensitivity into resistance surface construction to better represent environmental constraints in karst ecosystems.

### Improved Network Optimization Strategy

Developed an enhanced Low-Degree-First (LDF) optimization strategy by considering both network topology and environmental vulnerability.

### Robustness-Based Network Evaluation

Assessed ecological network resilience under random and targeted disturbances using robustness and recovery metrics.

## Methods

### Ecological Source Identification

- Morphological Spatial Pattern Analysis (MSPA)
- Habitat Quality Assessment (InVEST)
- Landscape Connectivity Analysis (dPC)
- Analytic Hierarchy Process (AHP)

### Resistance Surface Construction

- Land-use Resistance
- Topographic Resistance
- Human Disturbance Resistance
- Rocky Desertification Correction

### Ecological Corridor Extraction

- Minimum Cumulative Resistance Model (MCR)

### Network Analysis

- Degree Centrality
- Betweenness Centrality
- Edge Betweenness

### Network Optimization

- Improved Low-Degree-First (LDF) Strategy

### Robustness Assessment

- Random Attack Simulation
- Targeted Attack Simulation
- Network Recovery Analysis

## Key Results

| Indicator | Result |
|------------|------------|
| Ecological Sources | 79 |
| Ecological Source Area | 5,423 km² |
| Ecological Corridors | 91 |
| Corridor Length | 1,636 km |
| Added Corridors After Optimization | 27 |

### Main Findings

- Ecological sources are primarily distributed in northern and central mountainous regions.
- Rocky desertification significantly influences ecological connectivity patterns.
- Network optimization substantially improves connectivity and ecological flow.
- The optimized network exhibits higher robustness under both random and targeted disturbances.
